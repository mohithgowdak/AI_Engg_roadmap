# Module 4: Vector Stores & RAG

This module teaches you to build Retrieval-Augmented Generation (RAG) systems.

## Files in this Module

1. **10_vector_stores.py** - Working with Vector Stores
   - Chroma vector store
   - FAISS vector store
   - Adding documents
   - Querying vector stores
   - Metadata filtering

2. **11_retrieval_qa.py** - Retrieval-Augmented Generation
   - Basic RAG systems
   - Custom prompts for RAG
   - Source citations
   - Improving retrieval

## Learning Objectives

By the end of this module, you will:
- ✅ Store and search documents using vector stores
- ✅ Build RAG systems
- ✅ Answer questions using document context
- ✅ Improve RAG performance

## How to Run

```bash
python 10_vector_stores.py
python 11_retrieval_qa.py
```

## Prerequisites

- Completed Module 3: Document Processing
- OpenAI API key
