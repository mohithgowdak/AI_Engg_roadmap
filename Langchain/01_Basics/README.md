# Module 1: Basics

This module covers the fundamental concepts of LangChain.

## Files in this Module

1. **01_basic_llm.py** - Introduction to Language Models
   - Using OpenAI and Google Gemini models
   - Making simple LLM calls
   - Understanding temperature and model parameters

2. **02_prompts.py** - Prompts and Prompt Templates
   - Creating prompts
   - Using variables in prompts
   - Chat prompt templates
   - Dynamic prompt generation

3. **03_chains.py** - Creating and Using Chains
   - Simple chains (LLM + Prompt)
   - Sequential chains
   - Chain composition
   - Batch processing

4. **04_output_parsers.py** - Parsing LLM Outputs
   - Pydantic output parsers
   - List parsers
   - Structured output parsers
   - Error handling

## Learning Objectives

By the end of this module, you will:
- ✅ Understand how to use LLMs with LangChain
- ✅ Create and use prompt templates
- ✅ Build chains to combine components
- ✅ Parse and structure LLM outputs

## How to Run

Run each file in order:
```bash
python 01_basic_llm.py
python 02_prompts.py
python 03_chains.py
python 04_output_parsers.py
```

## Prerequisites

- Python 3.8+
- OpenAI API key (set in `.env` file)
- All dependencies from `requirements.txt`
