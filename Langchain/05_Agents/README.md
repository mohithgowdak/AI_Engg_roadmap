# Module 5: Agents

This module covers building intelligent agents that can use tools.

## Files in this Module

1. **12_agents_basic.py** - Introduction to Agents
   - Basic agent setup
   - Different agent types
   - Agent reasoning
   - Agents with memory

2. **13_agent_tools.py** - Creating Custom Tools
   - Simple tools
   - Tool decorators
   - Tools with Pydantic schemas
   - Multiple tools
   - Error handling

3. **14_agent_executor.py** - Advanced Agent Execution
   - Agent executors
   - Custom prompts
   - Callbacks
   - Error handling
   - Debugging

## Learning Objectives

By the end of this module, you will:
- ✅ Understand how agents work
- ✅ Create custom tools for agents
- ✅ Build agents that can reason and use tools
- ✅ Handle errors and debug agents

## How to Run

```bash
python 12_agents_basic.py
python 13_agent_tools.py
python 14_agent_executor.py
```

## Prerequisites

- Completed Module 1: Basics
- OpenAI API key
