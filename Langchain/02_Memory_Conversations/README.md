# Module 2: Memory & Conversations

This module teaches you how to build conversational AI with memory.

## Files in this Module

1. **05_memory_basic.py** - Basic Memory Concepts
   - ConversationBufferMemory
   - ConversationBufferWindowMemory
   - ConversationSummaryMemory
   - Memory operations

2. **06_conversation_chains.py** - Conversation Chains
   - Building chatbots
   - Contextual conversations
   - Specialized chatbots
   - Multi-turn conversations

## Learning Objectives

By the end of this module, you will:
- ✅ Understand different memory types
- ✅ Build chatbots with memory
- ✅ Maintain context across conversations
- ✅ Create specialized conversational agents

## How to Run

```bash
python 05_memory_basic.py
python 06_conversation_chains.py
```

## Prerequisites

- Completed Module 1: Basics
