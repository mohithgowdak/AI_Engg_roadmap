# Module 3: Document Processing

This module covers loading, processing, and understanding documents.

## Files in this Module

1. **07_document_loaders.py** - Loading Documents
   - Loading text files
   - Loading PDFs
   - Loading web pages
   - Loading from directories

2. **08_text_splitters.py** - Text Splitting
   - Character text splitter
   - Recursive character splitter
   - Token text splitter
   - Custom splitting strategies

3. **09_embeddings.py** - Understanding Embeddings
   - OpenAI embeddings
   - HuggingFace embeddings
   - Embedding documents
   - Similarity search

## Learning Objectives

By the end of this module, you will:
- ✅ Load documents from various sources
- ✅ Split documents into chunks
- ✅ Create embeddings for semantic search
- ✅ Find similar documents

## How to Run

```bash
python 07_document_loaders.py
python 08_text_splitters.py
python 09_embeddings.py
```

## Prerequisites

- Completed Module 1: Basics
- OpenAI API key
